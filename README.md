# 🚀 Olá, eu sou Lucas!

### 🎓 Estudante de Engenharia de Software
Sou apaixonado por transformar dados brutos em decisões estratégicas e por construir sistemas robustos no Back-end. Meu foco principal está na intersecção entre a **Análise de Dados** e o **Desenvolvimento de Software**, transitando entre a lógica de negócios e a infraestrutura de dados.

## 🛠️ No que eu trabalho
* **Data Analysis & BI:** Modelagem de dados, extração de insights e criação de dashboards para Inteligência de Negócios (BI).
* **Back-end Development:** Desenvolvimento de APIs e sistemas escaláveis com foco em performance e segurança.
* **Banco de Dados:** Gerenciamento e persistência de dados utilizando bancos relacionais.
* **Low-code & Mobile:** Desenvolvimento ágil de interfaces e aplicativos utilizando FlutterFlow.
* **Gestão Ágil:** Organização de fluxos de trabalho utilizando metodologias como Scrum e Kanban.
